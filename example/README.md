# sui_dart example

A minimal end-to-end example: create an account, request test SUI from the
devnet faucet, and read the resulting balance.

Run it from the package root:

```sh
dart run example/main.dart
```

See [`main.dart`](main.dart) for the full source, and the package
[README](../README.md) for more usage (transactions, signing, multisig,
zkLogin, and more).
